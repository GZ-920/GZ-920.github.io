## 源码

计算圆的面积

```cc
#include<iostream>
using namespace std;
#define pi 3.14
int main() {
	double r,s;
	cin >> r;
	s = pi * r*r; //运算符，下节课会讲
	cout << "圆的面积是：" << s;
	return 0;
}
```
