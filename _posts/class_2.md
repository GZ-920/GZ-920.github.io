# 课程源码

## 源码

```cc
#include<iostream>
using namespace std;

int main() {
	cout << "Hello World!" << endl;
	return 0;
}
```

## 注释源码 

```cc
#include<iostream> //输入输出头文件
using namespace std; //使用命名空间std

int main() { //定义主函数main
	cout << "Hello World!" << endl; //输出并换行
	return 0; //退出
} //main结束
```
